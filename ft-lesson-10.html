Every concept from lessons 01-09 assembled into one production pipeline. We fine-tune Llama-3-8B-model on a coding assistant task: curate data → SFT with QLoRA → evaluate → DPO alignment → merge → serve. This is exactly how production fine-tuning works at companies.

The Complete Pipeline

1
Curate & Filter Dataset
500-5000 high-quality (instruction, response) pairs. Apply quality filters. Apply chat template. 90/10 split.
2
QLoRA SFT Training
Load 4-bit Llama-3-8B-model. Add LoRA adapters (r=16). Train 3 epochs with cosine LR. ~90 min on T4.
3
Evaluate SFT Model
Run task-specific evals. LLM-judge on 100 test examples. Compare to base model. Decide if DPO is needed.
4
DPO Alignment (optional)
Generate 4 responses per prompt. Judge rankings. Train DPO for 1 epoch. Adds ~30 min.
5
Merge & Push to Hub
merge_and_unload(). Save in safetensors format. Push to private HF Hub repo for versioning.
6
Deploy with vLLM or Ollama
OpenAI-compatible API with vLLM. Local use with Ollama GGUF. Monitor latency and throughput.

"""
Fine-Tuning Pipeline: Llama-3-8B-model → Coding Assistant
Lessons 01-09 assembled into production-grade code.
Runs on free Colab T4 GPU (~2 hours total).
"""
import torch
from transformers import AutoModelForCausalLM, AutoTokenizer, BitsAndBytesConfig
from peft import LoraConfig, get_peft_model, prepare_model_for_kbit_training, PeftModel
from trl import SFTTrainer, SFTConfig, DPOTrainer, DPOConfig
from datasets import load_dataset, Dataset

MODEL_ID = "mistralai/Mistral-7B-Instruct-v0.3"  # no access request needed
SFT_OUT  = "./mistral-sft"
DPO_OUT  = "./mistral-dpo"
MERGED   = "./mistral-final"

# ── Step 1: Load in 4-bit ─────────────────────────────────────
bnb = BitsAndBytesConfig(load_in_4bit=True, bnb_4bit_quant_type="nf4",
                          bnb_4bit_compute_dtype=torch.bfloat16, bnb_4bit_use_double_quant=True)
model = AutoModelForCausalLM.from_pretrained(MODEL_ID, quantization_config=bnb,
          device_map="auto", torch_dtype=torch.bfloat16)
model.config.use_cache = False
model = prepare_model_for_kbit_training(model)
tokenizer = AutoTokenizer.from_pretrained(MODEL_ID)
tokenizer.pad_token = tokenizer.eos_token; tokenizer.padding_side = "right"

# ── Step 2: LoRA config ───────────────────────────────────────
lora = LoraConfig(r=16, lora_alpha=16, lora_dropout=0.05, bias="none",
                   task_type="CAUSAL_LM",
                   target_modules=["q_proj","k_proj","v_proj","o_proj"])
model = get_peft_model(model, lora)
model.print_trainable_parameters()

# ── Step 3: Load dataset ──────────────────────────────────────
raw = load_dataset("bigcode/self-instruct-starcoder", split="train[:3000]")
raw = raw.train_test_split(test_size=0.05, seed=42)

def format_example(ex):
    messages = [{"role":"user","content":ex["prompt"]},
                 {"role":"assistant","content":ex["completion"]}]
    return {"text": tokenizer.apply_chat_template(messages, tokenize=False)}

raw = raw.map(format_example)

# ── Step 4: SFT Training ──────────────────────────────────────
trainer = SFTTrainer(model=model, tokenizer=tokenizer,
    train_dataset=raw["train"], eval_dataset=raw["test"],
    args=SFTConfig(output_dir=SFT_OUT, num_train_epochs=3,
        per_device_train_batch_size=4, gradient_accumulation_steps=4,
        learning_rate=2e-4, bf16=True, optim="paged_adamw_32bit",
        gradient_checkpointing=True, max_seq_length=2048,
        logging_steps=25, save_strategy="epoch", eval_strategy="epoch",
        dataset_text_field="text", report_to="none"))
trainer.train()
trainer.save_model(SFT_OUT)
print("SFT complete. Adapter saved.")

# ── Step 5: Merge ─────────────────────────────────────────────
base = AutoModelForCausalLM.from_pretrained(MODEL_ID, torch_dtype=torch.float16, device_map="auto")
merged = PeftModel.from_pretrained(base, SFT_OUT).merge_and_unload()
merged.save_pretrained(MERGED, safe_serialization=True)
tokenizer.save_pretrained(MERGED)
print(f"Merged model saved to {MERGED}")

# ── Step 6: Quick sanity check ────────────────────────────────
merged.eval()
inputs = tokenizer(["[INST] Write a Python function to reverse a string. [/INST]"],
                    return_tensors="pt").to("cuda")
with torch.no_grad():
    out = merged.generate(**inputs, max_new_tokens=200, temperature=0.1, do_sample=True)
print(tokenizer.decode(out[0], skip_special_tokens=True))
